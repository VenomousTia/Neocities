Hello! You have opened werewolf-girlfriend's gallery template 1.0! This file (index.html) is a lil showcase of what the gallery is capable of, please take a look around :D You may also want to check out the blank version of this template in blank.html!

If you decide to use this template then I humbly request that you keep the disclaimer from the footer somewhere on the page <3 For more info on how to use this template, please read the HTML itself!
